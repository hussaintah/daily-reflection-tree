# Daily Reflection Tree

This repository contains a deterministic end-of-day reflection tool designed for the DeepThought Fellowship assignment.

The tool guides an employee through three psychological axes:

1. Locus: Victim vs Victor
2. Orientation: Contribution vs Entitlement
3. Radius: Self-Centrism vs Altrocentrism

The product does not use any LLM at runtime. The reflection experience is fully deterministic.
